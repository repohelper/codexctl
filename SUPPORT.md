# Support

## Where To Ask Questions

- GitHub Discussions (recommended for usage questions and ideas)
- GitHub Issues (for confirmed bugs and concrete feature requests)

## Before Opening An Issue

Please include:

- OS and architecture
- `codexctl --version`
- command executed
- full error output
- expected vs actual behavior

## Security Issues

Do not report security issues in public issues.
Use private disclosure:
https://github.com/repohelper/codexctl/security/advisories/new
